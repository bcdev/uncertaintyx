# Contributing

## Coding standards

Coding standards applied and adhered to in this project are based on [PEP&nbsp;8](https://pep8.org) 
guidelines. [Black](https://github.com/psf/black) or [Ruff](https://github.com/astral-sh/ruff) are used to
ensure that coding standards are met.

Tracing original mathematical notation in code is explicitly encouraged to enforce
readability and traceability, even if not compliant with [PEP&nbsp;8](https://pep8.org). For instance,
a single-character variable name may be capitalized if the variable is capitalized
in the original literature referred to.
